
Follow these instructions EXCEPT you don't need to use the imagecache 
url formatter anymore. Just pick any 'Image Style' formatter: http://drupal.org/node/758724 

To enable history:

1) Download and enable the Libraries API module 1.x version: http://drupal.org/project/libraries
2) Make a directory 'sites/all/libraries/history'
3) Download: https://galleriffic.googlecode.com/svn/trunk/example/js/jquery.history.js
4) Place the jquery.history.js filed in the 'sites/all/libraries/history' folder
